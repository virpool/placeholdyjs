placeholdyjs
============

Formatted placeholder for inputs

```html
<script src="../jquery.placeholdy.js"></script>

<input id="my-input" type="text" data-placeholdy-tmpl="template">
<script type="text/html" id="template">
    Blah <b>blah</b> &copy; 2013
</script>
```

```javascript
$('#my-input').placeholdy();
```
